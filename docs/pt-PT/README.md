# Linhas diretrizes para criar os guias de estilo da comunidade

Este documento fornece instruções 'passo a passo' para criar um guia de estilo comunitário. Os guias de estilo comunitários devem ser utilizados ​​na coordenação com o [Guia do estilo de localização geral da Mozilla] (../ mozilla_general/), mas fornece informação adicional vital para produzir uma tradução de alta qualidade no idioma da comunidade.

Para a criação de novos guias de estilo, nós preparamos um [modelo](outline.md) que pode utilizar. Cada cabeçalho no modelo representa um tópico de estilo comum na localização. Cada tópico apresenta uma ou mais perguntas que pertencem à informação do tópico. Por debaixo de cada pergunta está uma descrição de como responder à pergunta e alguns exemplos de informação relevante.

*Nota: Todos os guias de estilo utilizam a "Marcação", que já deve conhecer no Github. Se não, [Mastering Markdown](https://guides.github.com/features/mastering-markdown/) ou [Sintaxe básica de gravação e formatação](https://help.github.com/articles/basic-writing-and-formatting-syntax/) irá dar-lhe uma boa introdução.*

## Como utilizar estas linhas diretrizes

1. Criar uma pasta para o seu idioma em [/docs folder](https://github.com/mozilla-l10n/styleguides/tree/main/docs).
2. Copiar o ficheiro com o [modelo do guia de estilo](https://github.com/mozilla-l10n/styleguides/blob/master/docs/guidelines/outline.md) para a sua pasta do idioma.
3. Preencher cada secção do modelo, utilizando as questões e direções das linhas diretrizes em baixo. Consulte o [guia de estilo de exemplo](example_en-us.md) conforme necessário.
4. Submeter um pedido de envio aos administradores do repositório.
5. Solicitar uma revisão do seu guia de estilo de outro membro da sua comunidade (se não houver outros membros da sua comunidade, inclua uma nota no seu pedido de envio).

Se a sua comunidade já tiver um guia de estilo, poderá seguir estes passos, utilizando o seu guia de estilo como um recurso para obter informação relevante ou poderá utilizar estas diretrizes para rever e atualizar o seu guia de estilo.

O idioma que utiliza no seu guia de estilo é da sua comunidade. É recomendável que utilize o seu idioma ou o inglês (ou o seu idioma de origem se não traduzir do inglês).

Ao seguir estes passos, inclua apenas a informação relevante para o seu idioma. Se estiver a utilizar o modelo de guia de estilo, remova as seções do modelo que não se apliquem ao seu idioma ou localidade e remova os comentários do modelo incluído ao concluir as secções.

Depois de atualizar ou criar o seu guia de estilo comunitário, envie um pedido de envio aos gestores do repositório.
